Full featured Lightweight & Responsive Multi Task Plugin For WordPress.


== Description ==

The WP multi task plugin gives users the ability to create new posts, edit their profile all from the site frontend, so that the user does not need to enter the backend admin panel to do somthings.

WP multi task plugin allows you to display Frontend user profile and Frontend Login and Registration and Frontend Users Message system

<strong> Installation </strong>



1. Install as regular WordPress plugin.

2. Go your Pluings setting via WordPress Dashboard and active it.

3. After activate plugin you will see a new menu on the dashboard left side called WP Multi Task. 

<br /><br />



<strong>How to use WP Multi Task plugin page, post or widget.</strong><br />

* Frontend login shortcode [wpmt-login]
* Frontend Registration shortcode [wpmt-registration]
* Frontend Profile and message system [wpmt-profile]
